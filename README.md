# simon-says-game

This is a simple Simon Says memory game created using HTML, CSS, and JavaScript.
The game shows a sequence of colors, and the player has to repeat the same sequence.
With each correct round, the level increases and the sequence gets longer.

# How the Game Works

1.Press any key to start.

2.A random button will flash.

3.You have to click the buttons in the same order.

4.Each level adds one new color to the pattern.

5.If you click the wrong button, the game will show a Game Over message along with your score.

6.You can restart by pressing any key again.

# Features

1.Increasing difficulty with each level

2.Smooth flash animations on buttons

3.Clean and simple UI

4.Works without sound (animation-only game)
